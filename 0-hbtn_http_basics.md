# Basics of HTTP/HTTPS

## 1. Differences between HTTP and HTTPS
- **Security:** HTTP is unsecured and sends data in plain text. HTTPS is secure and encrypts data using SSL/TLS.
- **Port:** HTTP uses port 80. HTTPS uses port 443.

## 2. HTTP Request/Response Structure
- **Request:** Consists of a Method, Path, Version, Headers, and an optional Body.
- **Response:** Consists of a Version, Status Code, Status Message, Headers, and a Body.

## 3. Common HTTP Methods
- **GET:** Retrieves data.
- **POST:** Submits new data.
- **PUT:** Updates existing data.
- **DELETE:** Deletes data.

## 4. Common Status Codes
- **200 OK:** Success.
- **404 Not Found:** Resource not found.
- **500 Internal Server Error:** Server error.
