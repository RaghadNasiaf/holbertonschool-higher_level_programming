HTTP vs HTTPS:
- HTTP (Hypertext Transfer Protocol) transfers data in plain text, making it insecure.
- HTTPS (HTTP Secure) uses SSL/TLS encryption to protect data from tampering and eavesdropping.

HTTP Structure:
- Request: Method (GET, POST), Path, Headers, and Body.
- Response: Status Code, Headers, and Body.

Common HTTP Methods:
- GET: Retrieves data from a server (e.g., fetching a web page).
- POST: Submits new data to be processed (e.g., creating a profile).
- PUT: Updates an existing resource completely.
- DELETE: Removes a specific resource.

Common Status Codes:
- 200 OK: Request succeeded.
- 201 Created: Resource successfully created.
- 404 Not Found: Resource is not available on the server.
- 500 Internal Server Error: General server-side error.
