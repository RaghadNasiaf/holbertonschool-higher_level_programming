# Basics of HTTP/HTTPS
## 1. Differences between HTTP and HTTPS
- **Security:** HTTP sends data in plain text, while HTTPS uses SSL/TLS to encrypt data.
- **Port:** HTTP typically uses port 80, whereas HTTPS uses port 443.
- **Trust:** HTTPS requires a Digital Certificate to verify the website's identity.

## 2. HTTP Request/Response Structure
- **Request Line:** Method (GET, POST), Path, and Version.
- **Status Line:** HTTP Version and Status Code (e.g., 200 OK).

## 3. Common HTTP Methods
- **GET:** Retrieves data.
- **POST:** Submits new data.
- **PUT:** Updates existing data.
- **DELETE:** Removes a resource.

## 4. Common Status Codes
- **200 OK:** Request succeeded.
- **404 Not Found:** Resource doesn't exist.
- **500 Internal Server Error:** Server-side error.
