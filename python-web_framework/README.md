# Project: RESTful API
## Task 0: Basics of HTTP/HTTPS
- **HTTP**: Standard protocol for data transfer, not encrypted.
- **HTTPS**: Secure version of HTTP, uses SSL/TLS for encryption.
- **Methods**: GET (Retrieve), POST (Create), PUT (Update), DELETE (Remove).
- **Status Codes**: 200 (OK), 404 (Not Found), 500 (Server Error).
